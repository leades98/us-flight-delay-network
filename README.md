# us-flight-delay-network

Graph analysis of U.S. flight-delay propagation: airports as a network, with centrality, community detection, and diffusion simulation in R.

## Methods

Built the airport route network with igraph. Computed degree and eigenvector centrality to find high-impact hubs, ran community detection (spinglass, Louvain, Walktrap), and simulated delay propagation from seed airports.
